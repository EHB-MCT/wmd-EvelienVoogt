# Contributing Guidelines

Thank you for your interest in contributing to this project.

## Project Context
This project was developed as an individual academic assignment.
External contributions are not actively expected.

## Contributions
If you wish to experiment or build upon this project:
- Fork the repository
- Create a feature branch
- Follow existing code conventions
- Write clear commit messages

## Issues
Issues may be used to document bugs or ideas but are not guaranteed to be addressed.

## Code Style
Please follow the existing project structure and naming conventions.
